#flashcards/os/Unix 
***
***Расшифровка:*** Client [[Uniform Resource Locator (URL)|URL]].
Консольная [[Утилита|утилита]] для отправки запросов на [[Сервер (Клиент Серверная Архитектура)|серверы]] и скачивания данных.
- Проще - это браузер без графического интерфейса.
- Поддерживает десятки [[Протокол уровня|протоколов]], но почти всегда используется для работы с [[Протокол HTTP|HTTP]] и HTTPS.
- Нужна для проверки кода и отзывчивости сервера.
***
***Сценарии использования.***
1. Отправка HTTP-[[Методы (запросы) HTTP|запросов]] на сервер для проверки логики.
2. Скачивание файлов, [[Архивирование|архивов]], скриптов или картинок по прямой ссылке.
3. Проверка доступности сервера, его [[IP-адрес|IP-адреса]] и возвращаемых HTTP-[[Заголовок|заголовков]].
***
***Флаги.***
1. `-o` или `-O` - сохранение данных в файл.
2. `-I` - проверка [[Статусы (коды) HTTP|статуса]] ответа.
3. `-d` или `-X POST` - отправка `POST`-запроса.
4. `-H` - отправка JSON.
***
***Примеры.***
```bash
curl https://github.com
curl https://example.com -o my_site.html
curl -I https://example.com

curl -X POST https://mysite.com -d "username=ivan&password=123"

curl -X POST https://mysite.com \ -H "Content-Type: application/json" \ -d '{"player1_id": 42, "is_vs_computer": true}'
```
