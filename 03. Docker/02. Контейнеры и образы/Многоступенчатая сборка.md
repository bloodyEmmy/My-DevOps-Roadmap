Использование нескольких этапов сборки [[Образ контейнера (Docker Image)|образа]] в одном [[Dockerfile]].
- Каждый [[FROM команда Dockerfile|FROM]] создает отдельный этап, из одного этапа можно копировать артефакты в следующий этап.
- ***Основная цель*** - уменьшение размера итогового образа, оставляя только нужные файлы и зависимости (т.е. уменьшение количества [[Слой (layer)|слоев]]).
***
Использование нескольких `FROM` позволяет нам собрать один образ, состоящий из нескольких различных артефактов из [[Docker Hub, Registry|хаба]] для дальнейшей работы.
***Пример.***
Мы будем работать с JSON на Ubuntu. Тогда мы просто напишем:
```dockerfile
FROM node:18
FROM ubuntu:22.04
```
