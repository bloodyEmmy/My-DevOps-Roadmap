Уникальный SHA-256 хэш [[Образ контейнера (Docker Image)|образа]] или его [[Слой (layer)|слоев]]. 
Нужен для проверки того, что образ не поврежден и совпадает с оригиналом.
Образ можно запускать по `digest`:
```bash
docker run <образ>@sha256:94e814....
```
- Гарантирует, что запустится именно тот образ, чью контрольную сумму вписали.
***
***Синтаксис.***
```bash
docker images --digests
```
- Используется [[Images команда Docker|images]].
***
***Важно.***
1. Присваивается только при `pull / push` из / в [[Docker Hub, Registry|реестр]] образов.
2. Может равняться `<none>`, если образ собирался локально на хосте, а не был скачан с реестра. Т.е. помимо [[FROM команда Dockerfile|FROM]] были добавлены еще слои, например [[RUN команда Dockerfile|RUN]].
3. Не будет иметь сумму и тот образ, [[Dockerfile]] которого повторен в точности, поскольку сумма вычисляется по точному набору данных (прим: временные метки).
