#flashcards/Docker
***
Программное обеспечение, которое помогает собирать информацию о запущенных [[Контейнер|контейнерах]] и сервисах. 
- У [[Docker Daemon]] есть драйвер журналирования по умолчанию, но мы можем указать другой драйвер, и с этого момента контейнер будет использовать именно его.
- Драйвер журналирования по умолчанию - `json-file`.
- Можно переключить на базовый [[Journald (Systemd)|journald]] или [[Syslog|syslog]].
***
Проще говоря, это драйвер, который сохраняет [[Лог контейнера|логи]] в файлы JSON на хосте. 
Каждый контейнер имеет свой отдельный файл логов.
***
В каждом журнале содержится информация только об одном контейнере.
Узнать, какой драйвер выбран в системе хоста можно при помощи [[Info команда Docker|info]].
```bash
docker info --format '{{.LoggingDriver}}'
```
