#flashcards/Docker
***
Используется в Linux для просмотра и настройки маршрутизации (таблицы маршрутизации).
***Синтаксис.***
```bash
ip route <параметр>
```
- От параметров зависит действие команды.
***
***Параметры.***
1. Без параметра - просмотр маршрутов.
2. `show` - показать таблицу маршрутов.
3. `add` - добавить маршрут.
4. `del` - удалить маршрут.
5. `change` - изменить маршрут.
6. `flush` - очистить маршруты.
