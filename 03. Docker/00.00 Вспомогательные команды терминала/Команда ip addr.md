#flashcards
***
Используется для просмотра информации о сетевых интерфейсах и их адресах. Применяется в работе с [[Сети в Docker|сетями]], [[Virtual Ethernet Pair (Veth)|veth]] и др. 
Показывает интерфейсы системы, [[IP-адрес|IP-адреса]], [[MAC-адрес|MAC-адреса]] и др.
***
***Синтаксис.***
```bash
ip addr <swow> <interface_name>
```
- `show` - опциональная команда для вывода инфы о конкретном интерфейсе.
- `interface_name` - имя конкретного интерфейса.
***
***Примеры.***
```bash
ip addr
ip addr eth0
ip addr show vethd6c7895
```
