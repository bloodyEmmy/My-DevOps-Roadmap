#flashcards/Docker
***
- Задает основную команду, которая будет выполняться при запуске [[Контейнер|контейнера]]. По сути, это просто "точка входа" при запуске контейнера.
- Отличается от [[CMD команда Dockerfile|CMD]] тем, что первая - стабильная часть команды в [[Dockerfile]], фиксирующая исполняемый файл, а вторая - изменяемая часть.
***
***Синтаксис.***
```dockerfile
ENTRYPOINT ["/usr/local/.../docker-entrypoint.sh"]
```
Внутри квадратных скобок в кавычках пишется адрес до скрипта - т.е. команды.
***
Можно обойтись и без `ENTRYPOINT`.
```bash
docker run --entrypoint /bin/bash myimage
```
