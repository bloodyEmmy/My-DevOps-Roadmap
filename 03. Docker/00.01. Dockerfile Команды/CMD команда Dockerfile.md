#flashcards/Docker
***
Определяет команду ***по умолчанию***, которая выполняется при запуске контейнера. 
- Если при запуске ([[Run команда Docker|run]]) указывается другая команда - `CMD` полностью заменяется.
- Отличается от [[ENTRYPOINT команда Dockerfile|ENTRYPOINT]] тем, что вторая выполняется всегда при запуске контейнера. `CMD` можно заменить целиком.
***
***Важно.***
1. В [[Dockerfile]] может быть только один `CMD`. Если их несколько, то используется последний.
2. Не выполняется при сборке [[Образ контейнера (Docker Image)|образа]], а только при запуске [[Контейнер|контейнера]].
***
***Пример 1.***
```dockerfile
CMD ["echo", "привет"]
...
```
- И поменяем вывод при запуске.
```bash
docker run myimage             # привет
docker run myimage echo пока   # пока
```
- Здесь мы заменили `CMD`.
***
***Пример 2.***
```dockerfile
ENTRYPOINT ["echo"]
CMD ["привет"]
...
```
- `ENTRYPOINT` выполняется всегда.
```bash
docker run myimage        # привет
docker run myimage пока   # пока
```
