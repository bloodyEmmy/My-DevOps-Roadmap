#flashcards
***
Задает пользователя и группу, от имени которых будут выполняться команды в [[Контейнер|контейнере]].
***Важно.*** 
По умолчанию контейнер работает от ***root*** - обычного суперпользователя Linux.
***
***Синтаксис.***
```dockerfile
USER <имя>:<группа>
```
- `<имя>` - имя пользователя.
- `<группа>` - имя группы (опционально).
***
***Пример.***
```dockerfile
USER appuser:appgroup
USER root
USER appuser
```
