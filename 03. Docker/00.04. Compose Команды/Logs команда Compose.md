#flashcards/Docker
***
Показывает все [[Лог контейнера|логи]] сервисов.
***Синтаксис.***
```bash
docker compose logs <флаги> <сервисы> ...
```
- Можно вывести логи нескольких [[Контейнер|контейнеров]].
***
***Флаги.***
1. `--follow (-f)` - следить за логами в реальном времени.
2. `--tail N` - показать последние `N` строк логов.
3. `-t` - добавить временные метки.
