#flashcards/Docker
***
Собирает [[Образ контейнера (Docker Image)|образ]] из [[Dockerfile]], указанных при помощи [[Build раздел Compose|build]].
***Синтаксис.***
```bash
docker compose build <флаги> <сервис> ...
```
- Можно работать с несколькими сервисами за одну строку.
***
***Флаги.***
1. `--no-cache` - игнорирует [[Кэш в Docker|кэш]] при сборке.
2. `--pull` - вытягивает свежие образы базовых [[Слой (layer)|слоев]].
