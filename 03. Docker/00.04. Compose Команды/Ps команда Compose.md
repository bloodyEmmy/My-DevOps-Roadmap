#flashcards/Docker
***
Показывает список [[Контейнер|контейнеров]], запущенных через [[Docker Compose]].
***Синтаксис.***
```bash
docker compose ps <флаги>
```
***
***Флаги.***
1. `--all (-a)` - показывает все контейнеры - аналог [[Ps команда Docker|ps]].
2. `--quiet (-q)` - показывает только ID контейнеров.
