#flashcards/Docker
***
Позволяет работать с [[Том Docker|томами]].
***Синтаксис.***
```bash
docker volume <команда> <опции>
```
- `<команда>` - действие с томом.
- `<опции>` - доп параметры.
***
***Основные команды.***
1. `ls` - список всех томов.
2.  `create` - создать новый том
3. `inspect` - инфа о томе (как [[Inspect команда Docker|inspect]]).
4. `rm` - удаление.
