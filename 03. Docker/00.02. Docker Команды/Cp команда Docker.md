#flashcards
***
Позволяет копировать файлы как с хоста в [[Контейнер|контейнер]], так и наоборот - с контейнера на хост.
- Копировать можно как просто файлы, так и папки.
- Отличается от [[Bind Mount (Монтирование каталогов с хоста)|монтирования]] тем, что при реране контейнера все скопированные файлы внутри контейнера удалятся (решение - монтировать или прописывать добавление в [[Dockerfile]]).
***
***Синтаксис.***
```bash
docker cp <флаги> <источник> <назначение>
```
- На хосте: путь к файлу или директории.
- В контейнере: `имя:путь`.
***
***Хост -> Контейнер.***
```bash
docker cp test.txt my_container:/app/
```
- `test.txt` - копируемый файл (его полный адрес).
- `my_container` - контейнер.
- `/app/` - адрес в контейнере.
***ВАЖНО.***
В конце `app/` стоит `/`, поскольку это папка.
***
***Контейнер -> Хост.***
```bash
docker cp my_container:/app/test.txt ./test.txt
```
