#flashcards
***
Основная подкоманда для управлениями [[Контейнер|контейнерами]] в [[Docker]]. 
Работает аналогично [[Image команда Docker|image]], т.е. создана для разграничения возможностей работы - только с контейнерами.
***
***Синтаксис.***
```bash
docker container <команда> <флаги> <аргументы>
```
- `<команда>` - выполняемое действие.
***
***Команды.***
1. `ls` - список контейнеров. == `ps`.
2. `run` - [[Run команда Docker|создать и запустить]] контейнер.
3. `start` -  [[Start команда Docker|запустить остановленный]] контейнер.
4. `rm` - удалить контейнер.
5. `inspect` - подробная информация о контейнере.
6. `exec` - выполнить команду внутри работающего контейнера ([[Exec команда Docker|exec]]).
