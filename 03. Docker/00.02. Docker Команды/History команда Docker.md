#flashcards/Docker
***
Показывает [[Слой (layer)|слои]] [[Образ контейнера (Docker Image)|образа]] ***в хронологическом порядке, т.е. от последнего к базовому***.
Помогает понять , какие команды [[Dockerfile]] создали каждый слой и сколько каждый слой весит.
***
***Синтаксис.***
```bash
docker history <флаги> <образ>
```
***Флаги.***
1. `-q (--quiet)` - выводить только ID слоев.
2. `--no-trunc` - показывать полные значения (ID и команды), не усекать их.
