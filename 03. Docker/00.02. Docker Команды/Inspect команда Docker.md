#flashcards/Docker
***
Выводит подробную информацию о [[Контейнер|контейнере]], [[Образ контейнера (Docker Image)|образе]], томе или сети в формате ***JSON***.
- Возвращает конфигурацию контейнера, смонтированные тома, сетевые настройки, статус и состояние.
- Внутри JSON можно найти поле `.HostConfig.LogConfig`, содержащее информацию о [[Драйвер журналирования|драйвере журналирования]], который используется контейнером.
***
***Синтаксис.***
```bash
docker inspect <флаги> <объект> ...
```
- `<объект>` - имя или ID контейнера, образа, тома или сети.
- Можно указать несколько объектов через пробел.
