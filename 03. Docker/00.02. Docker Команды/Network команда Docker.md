#flashcards
***
Позволяет работать с сетями.
***Синтаксис.***
```bash
docker network <команда> <опции>
```
- `<опции>` - часто имя сети.
***
***Команды.***
1. `ls` - просмотр сетей.
2. `inspect <имя>` - детали в JSON (как в [[Inspect команда Docker|inspect]]).
3. `docker run --network <контейнер>` - подключение [[Контейнер|контейнера]] к сети при его запуске.
4. `docker network connect <сеть> <контейнер>` - подключение РАБОТАЮЩЕГО контейнера.
5. `docker network disconnect <сеть> <имя>` - отключение контейнера от сети
6. `rm <имя>` - удаление сети.
7. `--driver <драйвер>` - пишем используемый [[Собственные (встроенные) сетевые драйверы Docker|сетевой драйвер]], прим driver bridge.
