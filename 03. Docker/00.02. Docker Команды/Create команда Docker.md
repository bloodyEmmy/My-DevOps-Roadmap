#flashcards/Docker
***
Создает новый [[Контейнер|контейнер]] из [[Образ контейнера (Docker Image)|образа]], но не запускает его.
- Этим отличается от [[Run команда Docker|run]].
- Позволяет заранее подготовить контейнер с нужной конфигурацией и запустить его позже с помощью `docker start`.
***
***Синтаксис.***
```bash
docker create <флаги> <образ> <команда> <аргументы>
```
- `<образ>` - имя образа
- `<команда> <аргументы>` - опционально - команда, которая выполнится при запуске контейнера (через [[Start команда Docker|start]]).
***
***Флаги.***
1. `--name` - задать имя контейнера.
2. `-v <путь_хоста> <путь_контейнера>` - [[Bind Mount (Монтирование каталогов с хоста)|примонтировать]] [[Том Docker|том]] или директорию.
Флаги по типу `-d`, `-it` не применяются, поскольку контейнер еще не запущен.
