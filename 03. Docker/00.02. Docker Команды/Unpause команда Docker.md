#flashcards/Docker
***
Возобновляет выполнение процессов [[Контейнер|контейнера]], которые были приостановлены с помощью [[Pause команда Docker|pause]].
- Работает только для тех контейнеров, которые заморожены с помощью `pause`. 
- Если контейнер [[Stop команда Docker|остановлен]], то `unpause` не сработает.
***
***Синтаксис.***
```bash
docker unpause <контейнер>
```
- `<контейнер>` - имя или ID контейнера.
