#flashcards
***
Показывает список запущенных [[Контейнер|контейнеров]].
***Синтаксис.***
```bash
docker ps <флаги>
```
- В качестве вывода будут выведены ID контейнеров, [[Образ контейнера (Docker Image)|образы]], команда, время создания, статус, имя и др.
***
***Флаги.***
1. `-a (--all)` - показать все контейнеры, включая остановленные.
2. `-q (--quiet)` - вывести только ID контейнеров.
