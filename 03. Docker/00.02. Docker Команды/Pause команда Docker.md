#flashcards
***
Приостанавливает все процессы внутри запущенного [[Контейнер|контейнера]], не останавливая сам контейнер. 
Т.е. контейнер остается в памяти, но процессы внутри него замораживаются.
***
***Синтаксис.***
```bash
docker pause <контейнер>
```
- `<контейнер>` - его имя или ID.
***
***Интересный пример.***
```bash
docker pause $(docker ps -q)
```
- Используется [[Ps команда Docker|ps]].
