#flashcards
***
Общая команда для работы с [[Образ контейнера (Docker Image)|образами]]. 
- Она объединяет несколько подкоманд для управления образами: просмотр, удаление, создание, тегирования и т.д.
- Служит некоторым объединением команд для разграничения возможностей, т.е. применима только для образов. 
- Создана вместе с [[Container команда Docker|container]].
***
***Синтаксис.***
```bash
docker image <команда> <флаги> <аргументы>
```
- `<команда>` - конкретное действие с образом.
- `<флаги>` - дополнительные параметры (фильтры, формат вывода и тд.)
- `<аргументы>` - аргументы команды (имя образа, путь к [[Dockerfile]] ...)
***
***Команды.***
1. `ls` - список локальных образов.
2. `rm` - удаление образов.
3. `pull` - скачать образ с [[Docker Hub, Registry|хаба]].
4. `push` - отправить образ в реестр.
5. `build` - построить образ из Dockerfile. Аналог [[Build команда Docker|build]].
6. `inspect` - подробная информация. Аналог [[Inspect команда Docker|inspect]].
7. `save` - экспорт образа в tar-файл, чтобы его можно было перенести или сохранить вне [[Docker]].
