#flashcards/Docker
***
Удаляет [[Образ контейнера (Docker Image)|образ]] (один или несколько) из локального реестра [[Docker]].
***
***Синтаксис.***
```bash
docker rmi <флаги> <образ> ...
```
- `<образ>` - имя или ID образа.
- Можно указывать несколько образов через пробел.
***
***Флаги.***
1. `-f (--force)` - принудительно удалить образ (+ все [[Контейнер|контейнеры]], созданные на его основе), даже если он используется контейнерами.
2. `--no-prune` - не удалять промежуточные слои.
