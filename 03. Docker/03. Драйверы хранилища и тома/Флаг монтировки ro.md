#flashcards
***
***Расшифровка***: read-only - только для чтения.
Когда мы его используем вместе с [[Bind Mount (Монтирование каталогов с хоста)|биндом]], то [[Контейнер|контейнер]] может только читать файлы из папки, но не может ничего туда записывать или изменять.
***
***Синтаксис.***
```bash
docker run -v /home/user/data:/app/data:ro my-image
```
