#flashcards/Docker
***
Показывает [[Токен|токен]] и команду для присоединения [[Узлы в Swarm (Nodes)|узлов]] к [[Кластер Swarm|кластеру]].
***Синтаксис.***
```bash
docker swarm join-token <опции> <worker|manager>
```
- `worker` - показать токен для [[Рабочие узлы (worker nodes)|рабочего]].
- `manager` - показать токен для [[Управляющие узлы (manager nodes)|менеджера]].
***
***Флаги.***
1. `--rotate` - генерирует новый токен. Старый перестает работать. ***УЖЕ ПОДКЛЮЧЕННЫЕ УЗЛЫ РАБОТАЮТ КАК ПРЕЖДЕ.***
2. `--quiet (-q)` - вывести только токен без полной команды.
