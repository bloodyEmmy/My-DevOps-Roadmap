#flashcards/Docker
***
Список [[Задача (Task) в Swarm|задач]] (то есть [[Контейнер|контейнеров]]) конкретного [[Сервис в Swarm|сервиса]].
***Синтаксис.***
```bash
docker service ps <флаги> <сервис>
```
***
***Флаги.***
1. `--filter (-f) <ключ>=<значение>` - Фильтрация задач. Прим: `node=worker1`, `desired-state=running`...
2. `--quiet (-q)` - Только ID задач.
