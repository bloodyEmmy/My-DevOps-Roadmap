#flashcards/Docker
***
Изменяет конфигурацию существующего [[Узел в Swarm (Nodes)|узла]] внутри [[Кластер Swarm|кластера]].
Является одной из опций [[Node команда Swarm|node]].
***
***Синтаксис.***
```bash
docker node update <опции> <NODE>
```
- `<NODE>` - имя узла или его ID.
***
***Некоторые опции.***
1. `--availability <active|pause|drain>` - определяет один из параметров [[Виды статусов узлов Swarm|статуса узла]].
2. `--label-add <key>=<value>` - добавляет метку к узлу. Можно добавлять несколько меток за раз.
3. `--label-rm <key>` - удаляет [[Метка (Label)|метку]] с узла.
4. `--role <worker|manager>` - изменяет роль узла. Альтернатива `promote` / `demote`.
