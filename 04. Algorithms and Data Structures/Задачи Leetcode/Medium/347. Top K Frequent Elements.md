Дан массив целых чисел `nums`и целое число `k`. Верните наиболее `k` наиболее часто встречающихся элементов. Вы можете вернуть ответ в **любом порядке** .
***Пример.***
Входные данные: `nums = [1,2,1,2,1,2,3,1,3,2], k = 2`
Вывод: `[1,2]`

***
Я использовал для решения [[Хеш таблица|хеш-таблицу]] через [[Словарь в Python|словарь]]. По времени код выполняется за [[O-большое|O(n * log2 n)]] из-за долгого времени работы [[Sorted в Python|sorted]].
```python
class Solution:
    def topKFrequent(self, nums: List[int], k: int) -> List[int]:
        ans_hash = {}
        for value in nums:
            if value not in ans_hash:
                ans_hash[value] = 1
            else:
                ans_hash[value] += 1
        sorted_ans = sorted(ans_hash.items(), key=lambda x: x[1], reverse=True)
        ans_list = []
        for i in range(k):
            ans_list.append(sorted_ans[i][0])
        return ans_list
```
***Решение.***
1. Проходимся в первый раз по `nums`, если числа нет в словаре, то добавляем его, иначе - добавляем счетчик 1.
2. Сортируем словарь по значениям (а не по ключам, как обычно) при помощи [[Лямбда функция|лямбда функции]]. На выходе получаем список, элементами которого являются [[Кортеж в Python|кортежи]] формата `"(ключ, значение)"`.
3. Добавляем в итоговый массив первые `k` элементов отсортированного списка с индексом `0` - ключи.
