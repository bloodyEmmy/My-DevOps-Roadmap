Дана целочисленная матрица `nums`. Верните матрицу `answer`, равную `answer[i]` - произведению всех элементов массива `nums` за исключением `nums[i]` .
Произведение любого префикса или суффикса `nums`гарантированно **поместится** в **32-битное** целое число.
Вам необходимо написать алгоритм, который работает за  `O(n)` время и без использования операции деления.
***Пример 1.***
Входные данные: `nums = [1,2,3,4]`
Выходные данные: `[24,12,8,6]`
***Пример 2.***
Входные данные: `nums = [-1,1,0,-3,3]`
Выходные данные: `[0,0,9,0,0]`
#flashcards/Leetcode 
***
Для решения используются 2 списка, элементами которого являются произведения до конкретных `i`-ых элементов `nums`.
- Выполняется за время [[O-большое|O(n)]].
```python
class Solution:
    def productExceptSelf(self, nums: list[int]) -> list[int]:
        n = len(nums)
        res_from_left = [1] * n
        res_from_right = [1] * n
        for i in range(1, n):
            res_from_left[i] = res_from_left[i - 1] * nums[i - 1]
        for i in range(n - 2, -1, -1):
            res_from_right[i] = res_from_right[i + 1] * nums[i + 1]
        answer = [1] * n
        for i in range(n):
            answer[i] = res_from_left[i] * res_from_right[i]
        return answer
```
