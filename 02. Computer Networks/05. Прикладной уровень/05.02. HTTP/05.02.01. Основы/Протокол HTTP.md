#flashcards/network 
***
***Расшифровка:*** Hypertext Transfer Protocol.
***Перевод:*** [[Протокол уровня|Протокол]] передачи гипертекста.
- Основа [[World Wibe Web (WWW)|www]].
- Находится на [[Прикладной уровень|прикладном уровне]].
- Использует [[Протокол TCP|TCP]].
- Работает на основе [[Клиент Серверная Архитектура|клиент-серверной архитектуры]] по режиму работы "запрос - ответ".
- [[Порт]] по умолчанию - `80`.
***
Про структуру HTTP-пакета читаем [[Структура пакета HTTP|здесь]].
Имеет [[Методы (запросы) HTTP|методы]] и [[Статусы HTTP|статусы]].
Поддерживает [[Кэширование в HTTP|кэширование]].
