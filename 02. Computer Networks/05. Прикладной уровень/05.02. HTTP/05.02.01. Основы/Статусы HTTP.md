#flashcards/network 
***
Первое [[Структура пакета HTTP|поле]] в ответе [[Сервер (Клиент Серверная Архитектура)|сервера]].
1. `1xx` - информация.
2. `2xx` - успешное выполнение.
	- 200 - ОК.
3. `3xx` - перенаправление.
	- 301 - постоянное перемещение (страница перемещена на другой адрес навсегда).
	- 307 - временное перенаправление.
4. `4xx` - ошибка на стороне [[Клиент (Клиент Серверная Архитектура)|клиента]].
	- 403 - доступ запрещен.
	- 404 - страница не найдена.
5. `5xx` - ошибка сервера.
	- 500 - внутренняя ошибка сервера.
