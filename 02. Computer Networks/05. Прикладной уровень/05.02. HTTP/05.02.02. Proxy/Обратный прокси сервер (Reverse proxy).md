#flashcards/network 
***
[[Прокси сервер (Proxy)|Прокси]], который работает на стороне [[Сервер (Клиент Серверная Архитектура)|сервера]], обычный пользователь не знает о его существовании.
- Самый популярный софт для этого - Nginx.
***
***Роли.***
1. ***Балансировка нагрузки*** - можно поставить несколько одинаковых серверов с кодом, а перед ними - один прокси. Он принимает [[Протокол HTTP|HTTP]]-[[Методы (запросы) HTTP|запросы]] и равномерно распределяет между серверами.
2. ***Защита от DDoS*** - обратный прокси принимает удар на себя, фильтрует мусорные [[IP-адрес|IP-адреса]] и не пускает в систему.
3. ***Шифрование*** - для этого требуется работа [[Процессор (CPU)|процессора]], который жалко на классических серверах => прокси берет шифровку и дешифровку на себя.
