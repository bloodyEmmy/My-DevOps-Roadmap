#flashcards/network 
***
Используется на [[Прикладной уровень|прикладном уровне]] в [[Domain Name System (DNS)|DNS]].
- Работает по системе "запрос - ответ" по правилам [[Клиент Серверная Архитектура|клиент-серверной архитектуры]].
- Использует по умолчанию [[Порт|порт]] `53`.
- Работает при помощи [[Резолвер (Stub Resolver)|резолверов]], вшитых в [[Операционная система (ОС)|ОС]] рассматриваемых хостов.
- В качестве способа обмена данными использует [[Протокол UDP|UDP]]-пакеты.
***
***Что еще умеет?***
1. Определяет адреса в формате IPv4 и IPv6.
2. Задает несколько [[Домен|доменных имен]] для одного IP-адреса.
3. Находит адрес почтового [[Сервер (Клиент Серверная Архитектура)|сервера]] для домена.
4. Определяет IP-адрес и порт некоторых сетевых сервисов.
5. Задает адрес [[NS сервер (Name Server)|NS-серверов]] для доменных зон ([[Root уровень домена|Root]], [[TLD уровень домена|TLD]] или [[SLD уровень домена|SLD]]).
6. Определяет по IP-адресу доменное имя.
