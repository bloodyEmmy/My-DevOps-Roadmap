#flashcards/network 
***
Система доменных имен, которую проще назвать "телефонной книгой" интернета.
***Задача.***
Переводить понятные человеку буквенные адреса сайтов - [[Домен|домены]] (прим: `google.com`) в понятные компьютерам числовые [[IP-адрес|IP-адреса]] (при: `142.250.74.46`), которые нужны для сетевых [[Сокет (Socket)|сокетов]].
- Можно просмотреть IP-адреса при помощи [[Утилита|утилит]] `nslookup`, `host` или `dig`.
