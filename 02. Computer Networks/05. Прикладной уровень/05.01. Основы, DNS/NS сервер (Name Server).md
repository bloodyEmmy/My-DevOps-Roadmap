#flashcards/network 
***
Специализированный веб-сервер, который хранит главную "записную книжку" рассматриваемого [[Домен|домена]] (то есть [[Domain Name System (DNS)|DNS]]-зону) и сообщает интернету, на каком именно [[IP-адрес|IP-адресе]] находится сайт, почта или другие службы.
- То есть по сути, это конкретная железка, который хранит в себе текстовый файл с настройками уровня [[SLD уровень домена|SLD]].
- Именно он и отдает данные для того же DNS.
- В своих ответах содержит параметр [[TTL (Time To Live)|TTL]] - временной промежуток, который показывает, сколько по времени хранится ответ от NS-сервера у запрашиваемых ответ хостов (прим: [[Рекурсивный сервер|рекурсивный]] сервер).
***
***Какая информация на сервере.***
1. На каком IP-адресе работает сайт?
2. На какой [[Сервер (Клиент Серверная Архитектура)|сервер]] отправлять письма, если пишут на рассматриваемый адрес?
3. Какие у сайта есть [[Третий уровень домена (Поддомен)|поддомены]] и псевдонимы (пример: `://site.com`)?
4. Служебная информация (пример: подтверждение прав на владение сайтом для поисковиков).
