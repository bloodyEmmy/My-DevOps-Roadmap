#flashcards/network 
***
Встроенный системный компонент, находящийся в [[Операционная система (ОС)|ОС]].
- Windows - `dnsapi.dll`.
- Linux - часть библиотеки `glibc`.
- Именно через него работает вся система [[Domain Name System (DNS)|DNS]].
***
***Алгоритм работы.***
Когда любая программа ([[Процесс|процесс]]) хочет открыть сайт, то она делает [[Системный вызов|системный вызов]] к ОС и говорит, что ей нужен [[IP-адрес]] нужного [[Домен|домена]].
После этого:
1. Код [[Протокол уровня|протокола]] [[Протокол DNS|DNS]] выделяет в [[Оперативная память (ОЗУ)|ОЗУ]] [[Буфер|буфер]] под сетевой пакет.
2. Затем побайтово заполняет буфер по правилам стандарта DNS.
3. Вызывает функции [[Сетевая карта|сетевой карты]], чтобы завернуть этот буфер в [[Протокол UDP|UDP]]-пакет и отправить на адрес [[Сервер (Клиент Серверная Архитектура)|сервера]] (это может быть, например, [[Локальный DNS сервер|локальный]] или [[Открытый (Публичный) DNS сервер|открытый]]) через [[Порт|порт]] `53`.
