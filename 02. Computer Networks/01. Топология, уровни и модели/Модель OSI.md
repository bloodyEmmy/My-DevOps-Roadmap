#flashcards/network
***
Эталонная модель, описывающая:
1. Какие [[Уровни сети|уровни]] должны быть в сети.
2. Какие функции выполняются на каждом уровне.
Является [[Открытая система|открытой системой]].
***
***Свойства OSI.***
1. Не включает описание [[Протокол уровня|протокола уровня]].
2. 7 уровней.
3. На практике не используется.
4. Сильная теоретическая база.
***
***Уровни.***
1. ***[[Прикладной уровень|Прикладной]]*** - Набор приложений для работы пользователя (Web-страницы, социальные сети, видео / аудио связь, email).
2. ***Представления*** - Данные понятны отправителю и получателю (синтаксис, семантика, шифровка, дешифровка).
3. ***Сеансовый*** - Создает сеансы связи + очередность передачи сообщений + предотвращение одновременного выполнения.
4. ***[[Транспортный уровень|Транспортный]]*** - Обеспечивает передачу данных между [[Процесс|процессами]] на хостах + определяет порядок следования сообщений.
5. ***[[Сетевой уровень|Сетевой]]*** - Согласование различий в сетях. Определяет адрес + маршрут пересылки пакетов в [[Виды топологий сети|составной]] сети через узлы.
6. ***[[Канальный уровень|Канальный]]*** - Находит ошибки (+ корректирует), определяет начало и конец сообщения, находит получателя + определяет доступ. [[Ethernet]], [[Wi-Fi]].
7. ***[[Физический уровень|Физический]]*** - Передача битов по каналу связи (представляет биты в виде символов).
