#flashcards/os/Unix 
***
[[Утилита]], как и [[Top команда терминала|top]], являющаяся системным монитором, диспетчером задач [[Операционная система (ОС)|ОС]].
- Читает данные из [[Proc (VFS)|proc]].
***Отличия.***
1. По умолчанию нет на голой системе, нужно ставить через [[Команды для работы с пакетами|sudo apt install htop]].
2. Цветной интерфейс для отображения загрузки ядер [[Процессор (CPU)|процессора]] и [[Оперативная память (ОЗУ)|ОЗУ]].
3. Поддерживает мышь.
4. Показывает [[Процесс|процессы]] в виде деревьев.
