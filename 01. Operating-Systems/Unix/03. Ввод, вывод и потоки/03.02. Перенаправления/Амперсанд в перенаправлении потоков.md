#flashcards/os/Unix 
***
***Пример.***
Пусть я хочу, чтобы в моем скрипте все корректные результаты шли в файл `results.txt`, а строки с ошибками - в `errors.txt`. Для этого мне надо как-то разделить их между собой, так как обычный `echo` по умолчанию кидает все в [[Ввод, вывод, вывод ошибок|stdout]], то есть [[Терминал, Эмулятор терминала|терминал]] или один файл.
***
***Решение.***
Запускать скрипт мы будем следующей командой:
```bash
./script.sh argument 1> results.txt 2> errors.txt
```
Но тогда надо как-то в самом скрипте прописать условие вывода в различные файлы.
Выглядеть это будет так:
```bash
if [[ "$1" -ne "argument" ]]; then
	echo "Ошибка" >&2
	exit 1
fi
```
Таким образом, амперсанд непосредственно принимает участие в перенаправлении [[Перенаправление стандартного вывода|stdout]], [[Перенаправление стандартного вывода ошибок|stderr]] и вообще любого [[Файловый дескриптор (FD)|дескриптора]] (3, 4, 5...) рассматриваемого [[Процесс|процесса]].
