#flashcards/Python 
***
Анонимная функция, которая ***по умолчанию возвращает*** какое-то значение => писать `return` не нужно.
Используется, когда в качестве аргумента нужно использовать результат работы другой функции, чтобы не писать ее короткую реализацию.
***
***Пример 1.***
Базовое возведение в квадрат.
```Python
a = 5
square = lambda x: x**2
print(square(a)) # 25
```
***
***Пример 2.***
Изменить формат сортировки [[Словарь в Python|словаря]] по значениям по возрастанию вместо ключей (с использованием внутри [[Sorted в Python|sorted]]).
```
d = dict( smth )
sorted_d = dict(sorted(d.items(), key=lambda x: x[1], reverse=True))
```
