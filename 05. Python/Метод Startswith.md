Используется для проверки того, начинается ли строка с заданного набора символов или нет.
Разберем 2 примера: с поиском [[IP-адрес|IP-адресов]] и работой с [[Proc (VFS)|proc]].
***
***Пример 1.***
Пусть у нас есть IP-адрес. Нам нужно понять, находится ли он в текущей подсети (т.е. начинается ли его адрес с последовательности `192.168.x.x`).
```python
ip = "192.168.1.1"
if ip.startswith("192.168."):
	print("Локальный IP")
else:
	print("Внешний IP")
```

***Пример 2.***
Мы берем результаты данных по [[Оперативная память (ОЗУ)|ОЗУ]] и проверяем, есть ли еще место на ней. Вместо того, чтобы проверять все строки, выведенные через [[Cat команда терминала|cat]], руками, либо пробегать for-ом по всем строкам, можно выбрать только интересующие нас, а дальше не читать / редактировать.
```python
mem_total = 0
mem_available = 0

with open("/proc/meminfo", "r") as file:
	for line in file:
		if line.startswith("MemTotal: "):
			mem_total = int(line.split()[1])
		elif line.startswith("MemAvailable: "):
			mem_available = int(line.split()[1])
		if mem_total > 0 and mem_available > 0:
			break
```
