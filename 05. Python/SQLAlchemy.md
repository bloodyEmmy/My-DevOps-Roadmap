#flashcards/Python 
***
Программная библиотека языка Python, которая связывает код с [[СУБД (Системы Управления Базами Данных)|СУБД]].
- Переводит объекты и команды из кода Python на язык SQL.
- Является примером [[ORM (Object Relational Mapping)|ORM]].
***
***Применение.***
1. Автоматизация SQL-запросов  - не нужно писать запросы строками в Python, библиотека сама их сгенерирует.
2. Экранирует данные, защищая от [[SQL инъекция|SQL-инъекций]].
3. Превращает строки из таблиц СУБД в привычные объекты Python ([[Словарь в Python|словарь]], [[Массив|список]] и тд.).
4. Позволяет сменить СУБД не меняя код приложения (пример: с SQLite на [[PostgreSQL]]).
