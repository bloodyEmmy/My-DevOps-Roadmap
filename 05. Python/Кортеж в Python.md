Неизменяемая структура данных в Python. 
- В отличие от списка, ее невозможно модифицировать после создания.
- Из-за неизменяемости кортеж можно использовать как ключи для [[Словарь в Python|словарей]].
***Особенности.***
1. Нельзя добавить, поменять или убрать элемент.
2. Элементы хранятся в строгом порядке, заданном при инициализации кортежа.
3. Работает быстрее списков + занимает меньше места.
4. Защищает данные от изменений в коде.
***
***Примеры***.
```python
empty = ()
person = ("Алексей", 25, True)

single = (5,) # при задании кортежа из 1 элемента используем запятую

# операции
fruits = ("яблоко", "банан", "груша", "банан")
print(fruits[0]) # яблоко
print(fruits[1:3]) # банан груша
print(fruits.count("банан")) # 2
print(fruits.index("яблоко")) # 0
```
