#flashcards/databases  
***
Используется для упорядочивания результатов по одному или нескольким столбцам (прим: алфавит, дата, размер числа и др.).
- Всегда пишется в самом конце после [[WHERE оператор SQL|WHERE]].
- Направления сортировки: `ASC` (по умолчанию) - по возрастанию и `DESC` - по убыванию.
***
***Примеры.***
1. Сортировка по одному столбцу по возрастанию.
```SQL
SELECT name, age FROM person
ORDER BY age;
-- Выведет людей от самых молодых к старым
```
2. По убыванию.
```SQL
SELECT name, age FROM person
ORDER BY age DESC;
-- От старых к молодым
```
3. По алфавиту.
```SQL
SELECT name FROM person
ORDER BY name ASC;
-- Anna, Andrey, Boris...
```
***
***Нетривиальный пример.***
Если у людей одинаковый возраст, то SQL выводит их в случайном порядке => для исправления добавляем второй уровень сортировки для сортировки по убыванию.
```SQL
SELECT name, age FROM person
ORDER BY age ASC, name DESC;
```
