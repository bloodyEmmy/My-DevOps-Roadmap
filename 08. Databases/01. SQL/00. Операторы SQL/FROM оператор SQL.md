#flashcards/databases
***
Указывает базе данных, откуда брать информацию (какой именно источник / таблица).
- Обязательный спутник [[SELECT оператор SQL|SELECT]].
***
***Синтаксис.***
```SQL
SELECT ... FROM <имя источника>;
```
***
***Сценарии.***
1. Чтение из нескольких таблиц - устаревший способ объединения. Возвращает все возможные комбинации строк из двух таблиц.
```SQL
SELECT * FROM person, pizzeria;
```
2. Алиасы - если название таблицы длинное.
```SQL
SELECT p.name, p.age
FROM person p;
-- Буква "p" == "person"
```
