#flashcards/databases 
***
Отвечает за выбор нужных ***столбцов***.
- Не меняет данные в базе => только выводит их на экран в виде таблицы.
- Используется вместе с [[FROM оператор SQL|FROM]].
***
***Синтаксис.***
```SQL
SELECT <что ищем> FROM <название таблицы>;
```
***
***Сценарии.***
1. Выбрать все (все столбцы).
```SQL
SELECT * FROM person;
```
2. Выбрать конкретные столбцы - перечисление через запятую в том порядке, в котором они должны отобразиться.
```SQL
SELECT name, age FROM person;
```
