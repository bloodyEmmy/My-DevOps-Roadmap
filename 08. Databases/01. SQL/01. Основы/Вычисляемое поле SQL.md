#flashcards/databases  
***
Колонка в итоговой таблице, которой физически не существует в базе данных.
- Создается в момент выполнения запроса с помощью математических операций над существующими столбцами.
- Можно использовать `+`, `-`, `*`, `/` внутри оператора [[SELECT оператор SQL|SELECT]].
- Чтобы дать полю имя, используется [[AS оператора SELECT SQL|AS]].
***
***Примеры.***
1. Делаем скидку 100 рублей в стоимости пиццы.
```SQL
SELECT pizza_name, price - 100
FROM menu;
```
2. Даем новое имя.
```SQL
SELECT 
	pizza_name, 
	price * 0.9 AS discount_price
FROM menu;
```
