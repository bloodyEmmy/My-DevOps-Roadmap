#flashcards/databases 
***
***Перевод:*** Объектно-Реляционное отображение.
Технология программирования, которая позволяет работать с [[Реляционная база данных|реляционными базами данных]] через привычные объекты объектно-ориентированного языка.
- По сути, это переводчик между кодом и SQL, который позволяет не писать сложные запросы вручную.
- Защищает от [[SQL инъекция|SQL-инъекций]].
- Поддерживает переносимость с одной [[СУБД (Системы Управления Базами Данных)|СУБД]] в другую.
- Использует принцип [[Транзакция|транзакций]].
***
***Принцип работы.***
Вместо сложных запросов из [[SELECT оператор SQL|SELECT]], [[FROM оператор SQL|FROM]], [[WHERE оператор SQL|WHERE]] и других команд формата `SELECT * FROM users WHERE age > 18`, разработчик работает с объектами классов (пример: `User`).
- Таблицы == классы.
- Строки в таблице == объекты классов.
- Свойства (столбцы) == свойства объектов.
- Методы автоматически транслируются из кода в SQL-запросы.
